## 🎯 Game Mechanics

```

  Knight, Archer, Giant, Minions, Skeleton Army

BUILDINGS (2):
  Cannon, Bomb Tower

SPELLS (3):
  Fireball, Arrows, Freeze Spell
```

### Rarity Colors
```
🟢 Common (3 elixir)    → Knight, Archer, Cannon, Minions, etc.
🔵 Rare (4-5 elixir)    → Fireball, Baby Dragon, Valkyrie, etc.
🟣 Epic (6-7 elixir)    → P.E.K.K.A
⭐ Legendary (Special)  → Freeze Spell (animated gold)
```

### Difficulty Levels
```
🌱 Easy    → Random plays, slower pace, easy to beat
⚔️ Medium  → Balanced strategy, moderate difficulty
👹 Hard    → Aggressive plays, fast pace, challenging
```

## 🎮 How to Play

### Battle Screen
1. **Card Hand** (Bottom): Shows 4 cards, click to play
2. **Elixir Bar**: Purple bar shows current elixir
3. **Timer** (Top Center): 3 minutes total
4. **Tower HP** (Top): Shows health of all towers
5. **Arena** (Center): Where units battle

### Strategy
- Mix cheap and expensive cards
- Defend before attacking
- Use spells strategically
- Build up push momentum
- Counter enemy plays

### Win Conditions
- Destroy enemy King Tower
- Have more tower HP when time expires
- Win by tie-breaker (most tower damage)

## 💡 Pro Tips

### Building Decks
- **Balanced Mix**: Include 2-3 cheap cards, rest 4+ elixir
- **Card Types**: Mix troops, spells, and buildings
- **Synergy**: Cards that work well together
- **Average**: Aim for 3.5-4 average elixir cost

### Playing Matches
- **Early Game**: Build up elixir and defend
- **Mid Game**: Counter opponent's plays
- **Late Game**: Aggressive pushes for victory
- **Timing**: Don't waste elixir early

### Strategic Tips
- Watch opponent's card rotations
- Save elixir for good defenses
- Use spells strategically
- Build momentum with multiple cards
- Protect your towers!
