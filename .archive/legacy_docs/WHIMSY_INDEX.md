## 🔧 INTEGRATION POINTS (Where to Add Functions)

### In Victory Handler
```javascript
const handleVictory = () => {
  // Add this:
  triggerVictoryCelebration();
  const msg = getRandomMessage(PERSONALITY_MESSAGES.victory);
  // Display msg to player
}
```

### In Defeat Handler
```javascript
const handleDefeat = () => {
  // Add this:
  triggerDefeatWobble();
  const msg = getRandomMessage(PERSONALITY_MESSAGES.defeat);
  // Display msg to player
}
```

### When Unit Spawns
```javascript
const spawnUnit = (unit, index) => {
  // Add this:
  triggerUnitSpawn(unitElement, index);
}
```

### When Damage Is Dealt
```javascript
const damageUnit = (unit, damage) => {
  // Add these:
  triggerUnitHit(unitElement);
  showDamageNumber(damage, x, y);
}
```

### When Tower Falls
```javascript
const destroyTower = (tower) => {
  // Add this:
  triggerCrownDestruction(tower.x, tower.y);
}
```

## 🐛 TROUBLESHOOTING

### "Animations not showing"
→ Check CSS is in `<style>` tag (not external file)

### "Functions not working"
→ Verify functions are copied into component file

### "Performance drops"
→ Reduce confetti to 15-20 pieces

### "Mobile lag"
→ Use `WHIMSY_CSS_COPY_PASTE.css` instead of full version

### "Message not displaying"
→ Verify `getRandomMessage()` function is defined
