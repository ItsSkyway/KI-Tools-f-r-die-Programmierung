## 🚀 Deployment Guide

### Option 1: Direct (30 seconds)
```bash
1. Open index.html in browser
2. Start playing immediately
No installation, no setup needed!
```

### Option 2: Local Server
```bash
# Python
python -m http.server 8000
# Then visit: http://localhost:8000

# Node.js
npx http-server
# Then visit: http://localhost:8080

# Ruby
ruby -run -ehttpd . -p8000
# Then visit: http://localhost:8000
```

## 🎮 Game Overview

### How to Play
1. **Select Difficulty**: Easy 🌱, Medium ⚔️, Hard 👹
2. **Build Deck**: Choose 8 cards from 16 available
3. **Battle**: Play cards using elixir to defeat AI
4. **Win**: Destroy enemy King Tower or have more HP

### Strategy Tips
- Mix cheap and expensive cards
- Defend before attacking  
- Use spells strategically
- Build momentum with multiple units
- Counter enemy plays

### Card Types
- **Troops** (11): Ground and flying units
- **Buildings** (2): Defensive structures
- **Spells** (3): AOE effects and control
