🎯 COMBAT MECHANICS

SIMULTANEOUS COMBAT
  When 2 units in range:
  - Both attack in same frame
  - Damage applied simultaneously
  - No turn order

SWARM HANDLING
  Multiple units spawned:
  - Each is individual (not grouped)
  - Random loose formation
  - Each has own HP bar

BUILDING AUTO-ATTACK
  Buildings target automatically:
  - Nearest enemy selected
  - Auto-attack enabled
  - No manual targeting
  - Splash damage (60% falloff)

FREEZE EFFECT
  When frozen:
  - Can't move or attack
  - Duration: 2 seconds
  - Automatically expires

PERFORMANCE CAP
  Unit limit per side:
  - Max 50 (troops + buildings)
  - Oldest removed if exceeded
  - Maintains 30fps stable

🔍 DEBUGGING TIPS

Unit not attacking?
  ✓ Check distance < range
  ✓ Check cooldown elapsed
  ✓ Check frozen status
  ✓ Check HP > 0

Performance lag?
  ✓ Check unit count (F12)
  ✓ Should auto-cap at 50/side
  ✓ Look for excessive spawning

Building not attacking?
  ✓ Check building HP > 0
  ✓ Check enemies in range
  ✓ Check attack cooldown

Tower not attacking?
  ✓ Check tower HP > 0
  ✓ Check enemy in range (150px)
  ✓ Check 1.25s cooldown
