## 🚀 Usage Examples

### Play a Sound
```javascript
soundManager.playSfx('cardPlaced')
soundManager.playSfx('unitDeath')
soundManager.playSfx('victory')
```

### Create an Animation
```javascript
spriteAnimations.spawnUnitAnimation(unit)
spriteAnimations.killUnitAnimation(unit)
```

### Use CSS Animation
