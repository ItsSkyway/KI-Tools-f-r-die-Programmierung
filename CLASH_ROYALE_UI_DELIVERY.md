## 🎨 Design System Features

### Color Palette (Authentic Dark Fantasy)
```
Primary:      #9333ea (Purple borders)
Accent:       #a855f7 (Bright purple glow)
Player:       #10b981 (Green - success)
Enemy:        #ef4444 (Red - danger)
Warning:      #f59e0b (Yellow - caution)
Premium:      #fbbf24 (Gold - legendary)

Dark BG:      #0a0e27 (Midnight blue)
Darker BG:    #1a0a2e (Royal purple-blue)
Darkest BG:   #0f0420 (Almost black purple)
Panel:        rgba(31, 41, 55, 0.95) (Semi-transparent)
```

### Rarity System (4 Levels)
```
Common:       Brown gradient (#6B4423 → #C9A961)
Rare:         Blue gradient (#1E5F9E → #4A9FD8)
Epic:         Purple gradient (#6B2D7F → #A855F7)
Legendary:    Gold gradient (#B8860B → #FF8C00) with glow
```

### Typography System
```
Display Font: Arial Black (900 weight)
Body Font:    Arial (600-700 weight)

Sizes:
3.5rem  - VICTORY/DEFEAT titles
2.5rem  - Section headers
1.25rem - Buttons, important labels
1rem    - Body text
0.875rem - Card names, labels
0.75rem - Small text, badges
```

### Animation System
```
Timing:
Fast:   150ms (UI feedback)
Normal: 300ms (Interactive elements)
Slow:   500ms (Scene changes)

Easing: cubic-bezier(0.4, 0, 0.2, 1)
(Professional, smooth, natural motion)
```
